Risks and Technical Debts

  • Lack of adoption:: Adoption for OGC Standards in the Sensorweb is generally slow, especially integration into tools for data discovery and analysis. Currently there are no noticeable efforts to develop a frontend for the API.

This should be your motto for systematic detection and evaluation of risks and technical debts in the architecture, which will be needed by management stakeholders (e.g. project managers, product owners) as part of the overall risk analysis and measurement planning.
